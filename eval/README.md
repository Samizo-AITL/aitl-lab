# 評価データ・検証ログ

このディレクトリは、PoC設計における各種評価結果、テストログ、シミュレーションレポートなどを保存します。

---

## 内容例

- RTL・ソフトウェアのシミュレーション波形ファイル
- テストベンチ実行結果ログ
- 実機検証結果のトレース
- 評価用スクリプトやログ解析ツール

---

## 運用ルール

- 評価データは再現可能な形で保存
- 重要なログはバージョン管理システムにコミット推奨
- 不具合報告時は該当評価結果を添付
